# OpenStreetMap Map in React + TypeScript + Vite

This app displays an interactive OpenStreetMap using React, TypeScript, and Vite with Leaflet for map rendering.

## Features

- Interactive map centered on London
- OpenStreetMap tiles
- Built with React and TypeScript
- Fast development with Vite

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`
